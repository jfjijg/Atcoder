### 標準入力をtxtファイルに保存して簡単にする方法

プログラムに標準入力を提供する際に、コマンドラインから直接入力する代わりに、入力データをテキストファイルに保存してそのファイルを読み込む方法があります。これにより、複雑な入力データを簡単に管理できます。

#### 手順

1. **入力データをテキストファイルに保存する**

   例えば、以下のような入力データを `input.txt` というファイルに保存します。

   ```txt
   3
   1 8
   10 11
   21 5
   ```

2. **プログラムを実行する際にファイルから入力を読み込む**

   以下のコマンドを使用して、プログラムを実行し、`input.txt` から入力を読み込みます。

   ```bash
   python3 a.py < input.txt
   ```

   これにより、`a.py` プログラムは `input.txt` ファイルの内容を標準入力として読み込みます。

#### 例

以下は、`a.py` プログラムを `input.txt` ファイルから入力を読み込んで実行する例です。

```bash
python3 a.py < input.txt
```

この方法を使用することで、複雑な入力データを簡単に管理し、プログラムのテストを効率的に行うことができます。

### ディレクトリ構造

以下は、`Atcoder` ディレクトリの構造の例です。

```
Atcoder/
├── a.py
├── input.txt
└── readme.md
```

- `a.py`: プログラムのソースコード
- `input.txt`: 標準入力として使用する入力データを保存したファイル
- `readme.md`: この説明を含むドキュメントファイル
